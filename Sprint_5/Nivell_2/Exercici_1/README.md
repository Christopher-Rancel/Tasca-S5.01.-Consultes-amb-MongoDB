# Nivell 2 - Exercici 1

## Compte quants comentaris escriu un usuari/ària que utilitza "GAMEOFTHRON.ES" com a domini de correu electrònic.

---

## Descripció

En aquest exercici s’ha realitzat una consulta sobre la col·lecció `comments` utilitzant MongoDB Compass.  
L’objectiu és comptar quants comentaris han estat escrits per usuaris que tenen el domini de correu electrònic `GAMEOFTHRON.ES`.

---

## Codi

```
Filter: { email: /GAMEOFTHRON.ES/i }
```

---

## Resultat

El nombre de documents mostrats a MongoDB Compass correspon al total de comentaris realitzats per aquests usuaris.
